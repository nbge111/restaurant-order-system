# 🎮 多人在线五子棋游戏 - 使用说明

## 🚀 快速开始

### 方法1（推荐）：一键启动
1. 双击运行 `start-game-with-friends.bat`
2. 等待两个窗口都启动（游戏服务器 + ngrok）
3. 在 ngrok 窗口中找到 Forwarding 开头的公网地址
4. 将公网地址分享给朋友！

### 方法2：仅本地游戏
1. 双击运行 `start-game.bat`
2. 在浏览器中打开 http://localhost:3002
3. 适合自己测试或同一局域网内的朋友

---

## 📱 与朋友一起玩

### 1. 获取公网地址
启动 `start-game-with-friends.bat` 后，在 ngrok 窗口中会看到类似这样的地址：
```
Forwarding  https://abc123-def456.ngrok.io -> http://localhost:3002
```

复制那个 `https://...ngrok.io` 的地址，分享给你的朋友！

### 2. 朋友如何加入
- 朋友在浏览器中打开你分享的 ngrok 地址
- 输入昵称进入大厅
- 加入你创建的房间
- 开始对战！

---

## 🎯 游戏玩法

1. **进入大厅**：输入昵称后点击"进入游戏大厅"
2. **创建房间**：输入房间号点击"创建房间"
3. **加入房间**：点击房间列表中的房间加入
4. **开始游戏**：等待对手加入，黑棋先行
5. **下棋**：点击棋盘格子落子
6. **获胜**：率先连成五子的一方获胜！

---

## ⚙️ 注意事项

### ngrok 免费版限制
- 每次重启 ngrok，公网地址会变化
- 免费版有流量限制，但对于小游戏足够使用
- 连接可能不稳定，如断开请重新启动

### 网络要求
- 确保你的电脑可以访问外网
- 朋友需要能正常访问互联网

### 安全提示
- 分享的公网地址任何人都可以访问
- 建议只分享给认识的朋友
- 游戏结束后可以关闭 ngrok

## 🔧 常见问题

### Q: ngrok 窗口看不到地址？
A: 请等待 5-10 秒，ngrok 需要时间连接服务器

### Q: 朋友无法连接？
A: 
- 检查 ngrok 是否还在运行
- 确认地址是否复制完整（包括 https://）
- 尝试重新启动 ngrok 获取新地址

### Q: 游戏断开了？
A: 刷新页面重新连接即可

### Q: 想换端口？
A: 编辑 `board-game-app/server.js`，修改 `const PORT = 3002;` 这一行

## 💡 高级用法

### 想在局域网内玩？
- 不用 ngrok，直接用你的局域网 IP
- 例如：http://192.168.1.100:3002
- 朋友需要和你在同一个 WiFi 网络

### 想部署到云服务器？
- 将 board-game-app 文件夹上传到云服务器
- 在服务器上运行 npm install 和 node server.js
- 配置安全组开放 3002 端口
- 分享你的服务器 IP 地址
